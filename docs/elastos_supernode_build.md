# Elastos Supernode Build

## System Requirements

- **RAM**: 1 GB
- **HDD**: 8 GB
- **OS**: Ubuntu 16.04 x86_64

## Building

```bash
$ cd $HOME
$ git clone https://github.com/elastos/Elastos.ELA.Supernode

$ cd $HOME/Elastos.ELA.Supernode/build
$ ./build.sh

$ ls -l ~/Elastos.ELA.Supernode/release
```
